# Інструкція з додавання нових соцмереж та проєктів

## Додавання нової соцмережі

1. **Додайте іконку** в папку `src/assets/` (наприклад, `instagram.png`).
2. **Відкрийте файл** `src/components/Social.jsx`.
3. **Знайдіть масив `items`** (приблизно рядок 5-10). Додайте новий об'єкт у масив:

```js
{ icon: '/src/assets/instagram.png', name: 'Instagram', url: 'https://instagram.com/vortexabra' },
```

4. **Збережіть файл**. Зміни застосуються автоматично.

## Додавання нового проєкту

1. **Додайте зображення** в папку `src/assets/` (наприклад, `new-game.png`).
2. **Відкрийте файл** `src/components/Projects.jsx`.
3. **Знайдіть масив `projects`** (приблизно рядок 5-15). Додайте новий об'єкт у масив:

```js
{
  title: 'Назва проєкту',
  descKey: 'projects.newgame', // ключ для перекладу
  link: 'https://...',
  linkKey: 'projects.newgame.link', // ключ для тексту кнопки
  img: '/src/assets/new-game.png',
},
```

4. **Додайте переклади** у файл `src/i18n.js`:

- Для української (в об'єкті `uk`):
```js
'projects.newgame': 'Опис проєкту українською.',
'projects.newgame.link': 'Детальніше',
```

- Для англійської (в об'єкті `en`):
```js
'projects.newgame': 'Project description in English.',
'projects.newgame.link': 'Learn More',
```

5. **Збережіть обидва файли**. Зміни застосуються автоматично.

## Примітки

- Усі шляхи до зображень мають починатися з `/src/assets/`.
- Назви файлів мають бути у нижньому регістрі, без пробілів (дефіси замість пробілів).
- Після додавання файлів перезапустіть сервер розробки (`npm run dev`), якщо зміни не відображаються.

## Як змінити рекламу

1. **Зображення** знаходиться в `src/assets/ad-placeholder.png`.
2. **Заміна зображення**: замініть файл `ad-placeholder.png` на новий рекламний банер. Рекомендовані розміри: 800×200 пікселів (або пропорційно). Файл має бути у форматі PNG.
3. **Зміна тексту або посилання**: відкрийте файл `src/components/AdSection.jsx`. Знайдіть рядок з текстом «Щоб замовити рекламу, перейдіть у наш» та посиланням. Змініть текст або URL за потреби.
4. **Приклад зміни тексту**:
   ```jsx
   <p className="ad-section__text">
     Щоб замовити рекламу, напишіть нам на{' '}
     <a href="https://t.me/vortexabra" target="_blank" rel="noopener noreferrer">
       Telegram
     </a>.
   </p>
   ```
5. **Приклад зміни зображення**: просто замініть файл `ad-placeholder.png` на інший з такою ж назвою або змініть шлях в імпорті в `AdSection.jsx`.
